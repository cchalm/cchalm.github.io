The Basics

Every Java program needs to have a public class with a method called "main". Here is a very simple Java program that prints the string "Hello, world!". 

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, world!");
    }
}

Take note of the main method. It has to be defined exactly like this: public static void main(String[] args).

Methods

"main" is a method that automatically gets called when you run the program; you can define your own methods too. Here is an example of a method called "printWildcats" being called twice in main. 

public class HelloWorld {
    public static void main(String[] args) {
      printWildcats();
      printWildcats();
    }

    public static void printWildcats() {
        System.out.println("Dem cats be wild");
    }
}

Note the syntax for defining a method. It's the same for main and printWildcats: 

public static void <method name>() {
  <methodbody>
}
